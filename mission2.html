Mission 2

Contexte:

Dans la mission 2, l'objectif était de commencer à bien exploiter la base de données de Kdou Business et comprendre le lien entre les tables à partir de l'interface QBE (Query By Example). QBE est une interface low code permettant l'exécution de requêtes SQL sous forme de tableau et de schéma de tables. Cela permet d'avoir un aspect visuel sur les liens des tables, comment sont-elles reliées ? Par quoi ? etc'. Il permet ensuite d'afficher le résultat sous forme SQL et donc cela nous apprend avec la pratique à quoi correspond chaque commande comme par exemple la catégorie critères qui correspond au WHERE en SQL. Nous avons mis quelques requête que nous avons effectuer dans cette mission.
